Welcome to Social Media Video API

About the API

This API allows you to extract downloadable URLs for public videos and images from social media platforms like Instagram, Facebook, Twitter/X, Pinterest, and more. It's powered by yt-dlp for reliable extraction.

How to Use

  • Endpoint: /get-video-url
  • Method: GET
  • Param: video_url (string, required) - The social media link (e.g., https://www.instagram.com/reel/example/)
  • Example: /get-video-url?video_url=https://www.instagram.com/reel/DMbsVhlTwAr/?igsh=OTh4b3U4Y3Y3a2Vu
  • Response: JSON with "download_url", "title", and optional "audio_url"

For more details, check the source code on GitHub or contact the developer.

About Our Website

The accompanying website is a user-friendly interface for this API. Paste a link, click download, and save videos/images directly. It's responsive for mobile and desktop, with modern UI features like previews and status updates. Visit our site to try it out.

Features

  • Supports multiple platforms
  • No login required for public content
  • High-quality downloads
  • Progressive Web App (PWA) support for installable app experience
